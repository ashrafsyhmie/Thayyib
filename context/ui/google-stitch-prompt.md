The platform helps factories manage supplier certificates, upload compliance documents, analyze ingredient lists using AI, track certificate expiry, and prepare for halal audits.

Style:
- Clean enterprise SaaS
- Professional and trustworthy
- Light mode
- Emerald green accents
- Sidebar navigation
- Card-based dashboard
- Data tables
- Responsive desktop-first design

Screens:
1. Login
2. Dashboard overview
3. Supplier management
4. Document management
5. AI analyzer
6. Audit readiness center
7. Notifications
8. Settings

Dashboard should show:
- Halal compliance score
- Expiring certificates
- AI risk alerts
- Recent uploads
- Audit readiness progress

Make it look like a serious B2B SaaS product, not a student project.
