# Design System

## Design Direction

Enterprise SaaS, modern, clean, trustworthy.

## Visual Identity

Name: Thayyib

Tone:
- Trustworthy
- Calm
- Professional
- Intelligent
- Compliance-focused

## Colors

Primary:
- Emerald / Green

Secondary:
- White
- Slate
- Soft gray

Status:
- Green: compliant
- Yellow: expiring soon
- Red: risk
- Blue: information

## Typography

Use modern sans-serif font.

Recommended:
- Inter
- Geist
- Plus Jakarta Sans

## UI Style

- Sidebar navigation
- Card-based dashboard
- Data tables
- Clear filters
- Document upload zones
- Status badges
- Audit progress bars
