# Authentication and RBAC

## Roles

Admin:
- Manage company settings
- Manage users
- Access all modules

Compliance Officer:
- Manage suppliers
- Upload documents
- Run AI analysis
- Prepare audits

Manager:
- View dashboard
- View reports
- Approve actions

Auditor:
- Read-only access to selected audit evidence

## MVP Auth

Use JWT or Supabase Auth.

For beginner-friendly build, Supabase Auth is acceptable.
