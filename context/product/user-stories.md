# User Stories

## Supplier Tracking

As a compliance officer, I want to add suppliers so that I can track their halal certification status.

## Document Upload

As a compliance officer, I want to upload certificates and SOP documents so that they are stored in one place.

## AI Analyzer

As a QA manager, I want the system to analyze ingredient lists so that risky ingredients can be flagged early.

## Dashboard

As a factory manager, I want a compliance dashboard so that I can understand audit readiness quickly.

## Reminders

As a compliance officer, I want expiry reminders so that I do not miss renewal deadlines.
