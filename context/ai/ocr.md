# OCR

## Goal

Convert uploaded PDFs/images into text.

## Options

Beginner:
- Tesseract OCR

Better accuracy:
- Google Vision API
- Azure Document Intelligence

## MVP Recommendation

Start with text PDFs and image OCR using Tesseract.

Add cloud OCR later if needed.

## OCR Challenges

- Blurry scans
- Tables
- Stamps
- Malay/English mixed text
- Arabic terms
