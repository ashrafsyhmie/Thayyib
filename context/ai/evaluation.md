# AI Evaluation

## Why Evaluation Matters

Compliance software must be trustworthy.

The AI should be tested regularly.

## Evaluation Criteria

- Correctly identifies risky terms
- Avoids false certainty
- Provides explanation
- Uses sources
- Recommends human verification
- Does not hallucinate regulations

## Test Cases

1. Gelatin with no source
2. Pork-derived term
3. Valid supplier certificate
4. Expired certificate
5. Clean ingredient list
6. Unclear OCR extraction
