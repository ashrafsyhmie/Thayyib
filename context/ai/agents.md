# Agentic AI Design

## Why Agentic AI

Modern enterprise AI is moving toward task-specific agents instead of one generic chatbot.

For Thayyib, each agent should have a narrow responsibility.

## Proposed Agents

### Document Agent
Classifies uploaded documents and extracts metadata.

### Ingredient Agent
Analyzes ingredients and E-numbers for potential halal risks.

### Supplier Agent
Checks supplier certificate status and expiry.

### Audit Agent
Evaluates audit readiness and missing evidence.

### Reporting Agent
Generates summaries and compliance reports.

## MVP Implementation

Do not build autonomous agents first.

Implement them as service modules.

Later, they can become real orchestrated agents.
